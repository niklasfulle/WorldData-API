## Continents

### Description

List of all continents in the world. The list is sorted alphabetically by continent name.

### Data Model

```md filename="Version: V4" copy
{
  "id": Number,
  "name": String,
  "area_km2": Number,
  "population": Number,
  "density_km2": Number,
  "countries": [
    {
      "id": Number,
      "name": String,
    }
  ]
}
```
