## Continents

### Functions

Returns all continents: <br/>
`/api/v1/continents`

Will return a `200` status code

Returns a specific continent: <br/>
`/api/v1/continents/name/[name]`

Will return a `200` status code if the continent exists, otherwise a `404` status code

## Oceans

### Functions

## Seas

### Functions

Returns all seas: <br/>
`/api/v1/seas`

Returns a specific sea: <br/>
`/api/v1/seas/name/[name]`

<Callout> Spaces in the name are replaced with a `-` for the search</Callout>
