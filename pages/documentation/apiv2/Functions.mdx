## Continents

### Functions

Returns all continents: <br/>
`/api/v1/continents`

Will return a `200 OK` status code

Returns a specific continent: <br/>
`/api/v1/continents/name/[name]`

Will return a `200 OK` status code if the continent exists, otherwise a `404 Not Found` status code
