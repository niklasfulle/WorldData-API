## Continents

### Description

List of all continents in the world. The list is sorted alphabetically by continent name.

### Data Model

```md filename="Version: V1" copy
{
  "id": Number,
  "name": String,
  "area_km2": Number,
  "population": Number,
  "density_km2": Number
}
```

## Countries

### Description

List of the 195 United Nations member states in the world. The list is sorted alphabetically by country name.

### Data Model

```md filename="Version: V1" copy
{
  "id": Number,
  "name": String,
  "iso3": String,
  "iso2": String,
  "numeric_code": Number,
  "phone_code": String,
  "capital": String,
  "tld": String,
  "region": String,
  "subregion": String,
  "latitude": String,
  "longitude": String,
  "independent": Boolean
}
```
